What is ngjs-color-picker?

A small directive which essentially is a color picker but with a few extra functions - for AngularJS.

On this page you can see examples, documentation and usage of the directive.

Available options

Selected color

JS Attribute: selectedColor
HTML Attribute: selected-color
Description: The default selected color in the color picker.
Default: First element in array of included colors.

Custom colors

JS Attribute: customColors
HTML Attribute: custom-colors
Description: Custom colors for the color picker.
Default: An array with following elements: [ #7bd148, #5484ed, #a4bdfc, #46d6db, #7ae7bf, #51b749, #fbd75b, #ffb878, #ff887c, #dc2127, #dbadff, #e1e1e1 ]

Options

JS/HTML Attribute: options
Description: An object containing several options for the color picker.

Keys in options

Key: roundCorners
Description: Option for round corners.
Default: false

Key: size
Description: Option for size in pixels of each color box in the color picker.
Default: 20

Key: total
Description: Option for total colors visible.
Default: Length of color array (default color array has 12 elements)

Key: horizontal
Description: Option for horizontal or vertical color picker.
Default: true

Key: columns
Description: Option for columns and how many.
Default: 0
Note: An even color picker with columns can only be created if the total can be divided with columns with no remainder (total % columns === 0)

Gradient

JS/HTML Attribute: gradient
Description: An object containing options for creating gradient colors.

Keys in gradient

Key: count
Description: Option for how many colors in the gradient.
Default: 10
Note: 14 is maximum.

Key: start
Description: Option for the starting color, must be full hex (either #08a35c or 08a35c).
Note: Must be included for gradient to work.

Key: step
Description: Option for how "fast" the gradient will change color.
Default: 1

Priorities

The directive will prioritize accordingly

  1. Custom colors
  2. Random colors
  3. Gradient
  4. Default colors

Basic usage example



Code

Example with larger size and round corners



Code

// HTML // JS $scope.customOptions = { size: 30, roundCorners: true }

Example with random colors



Code

// HTML // JS $scope.optionsRandom = { randomColors: 10 }

Example with gradient



Code

// HTML // JS $scope.optionsGradient = { start: '#BA693E', count: 10, step: 1 }

Example with vertical color picker



Code

// HTML // JS $scope.optionsVertical = { horizontal: false, total: 5 }

Example with columns defined



Code

// HTML // JS $scope.optionsColumn = { columns: 4, roundCorners: true }
